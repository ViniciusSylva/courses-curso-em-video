# 📚 Cursos - Curso em Vídeo

Este repositório reúne exercícios, desafios e projetos desenvolvidos durante os cursos da plataforma **Curso em Vídeo**, ministrados pelo professor **Gustavo Guanabara**.

O objetivo deste repositório é organizar os conteúdos estudados e acompanhar a evolução durante o aprendizado em programação e desenvolvimento web.

---

## 📂 Cada pasta contém:

- 📖 exercícios realizados durante as aulas
- 🧩 desafios propostos nos cursos
- 💻 pequenos projetos desenvolvidos durante o aprendizado
